# Data Structures and Algorithms Syllabus

This syllabus is organized into 8 classes. Each class has a theory focus, a practical implementation focus, and LeetCode practice problems.

## Class 1 - Arrays & Complexity

### Theory

- Big O notation for time complexity and space complexity.
- Static arrays vs dynamic arrays.
- Array memory layout and why index access is O(1).
- Two-pointer pattern.
- Sliding window pattern.


### LeetCode Practice

- Two Sum
- Best Time to Buy and Sell Stock
- Longest Substring Without Repeating Characters

## Class 2 - Linked Lists

### Theory

- Singly linked lists and doubly linked lists.
- Pointer manipulation.
- Floyd's cycle detection with fast and slow pointers.
- When to prefer linked lists over arrays.

### Implementation

- Build a singly linked list from scratch.
- Reverse a linked list iteratively and recursively.
- Detect and remove a cycle.

### LeetCode Practice

- Reverse Linked List
- Merge Two Sorted Lists
- Linked List Cycle II

## Class 3 - Stacks & Queues

### Theory

- Stack LIFO behavior and call stack intuition.
- Queue FIFO behavior and its connection to BFS.
- Monotonic stack pattern.
- Deque and sliding window queue.

### Implementation

- Build a stack with an array and with a linked list.
- Build a queue using two stacks.
- Solve next greater element with a monotonic stack.

### LeetCode Practice

- Valid Parentheses
- Min Stack
- Daily Temperatures

## Class 4 - Hash Maps & Sets

### Theory

- Hash functions and collision resolution.
- Load factor and rehashing.
- Frequency counting pattern.
- Complement and group-by patterns.

### Implementation

- Build a hash map with chaining.
- Group anagrams.
- Preview LRU Cache with OrderedDict.

### LeetCode Practice

- Valid Anagram
- Group Anagrams
- Top K Frequent Elements

## Class 5 - Trees & Recursion

### Theory

- BST properties and invariants.
- DFS traversal: preorder, inorder, and postorder.
- BFS traversal with a queue.
- Recursion vs iteration trade-offs.

### Implementation

- Implement BST insert, search, and delete.
- Implement all three traversals recursively and iteratively.
- Implement level-order traversal with BFS.

### LeetCode Practice

- Maximum Depth of Binary Tree
- Validate Binary Search Tree
- Binary Tree Level Order Traversal

## Class 6 - Graphs

### Theory

- Adjacency list vs adjacency matrix.
- BFS for shortest path in unweighted graphs.
- DFS for connected components and cycle detection.
- Topological sort with Kahn's algorithm.

### Implementation

- Build a graph from an edge list.
- Implement BFS shortest path.
- Implement DFS cycle detection in a directed graph.

### LeetCode Practice

- Number of Islands
- Clone Graph
- Course Schedule

## Class 7 - Sorting & Searching

### Theory

- Merge sort and quick sort with divide and conquer.
- Heap sort and the priority queue.
- Binary search and its variants.
- Sort-first strategies paired with two pointers.

### Implementation

- Implement merge sort from scratch.
- Implement binary search on a rotated sorted array.
- Find the k-th largest element using a heap.

### LeetCode Practice

- Binary Search
- Search in Rotated Sorted Array
- Find Median from Data Stream

## Class 8 - Dynamic Programming

### Theory

- Overlapping subproblems and optimal substructure.
- Top-down memoization vs bottom-up tabulation.
- 1D DP: Fibonacci, climbing stairs, and house robber.
- 2D DP: knapsack and longest common subsequence.

### Implementation

- Implement Fibonacci from brute force to memoization to tabulation.
- Implement 0/1 knapsack bottom-up.
- Implement longest common subsequence.

### LeetCode Practice

- Climbing Stairs
- House Robber
- Longest Common Subsequence
